# App Shell Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ввести единый закреплённый desktop top bar (поиск, уведомления, avatar-меню), очистить sidebar до одних разделов, сделать правый рейл collapsible — не сломав mobile и messenger.

**Architecture:** Всё крепится в `AppLayout`. Новый `DesktopTopBar` — первый flex-ребёнок desktop-шелла (фикс. высота), строка колонок ниже — `flex-1`; бар закреплён естественно (шелл `overflow-hidden`, скроллится только `<main>`). `UserMenu` — dropdown на UI Kit с hover+click. Sidebar теряет аккаунт-хром. RightCategories получает collapse с localStorage. Messenger пересчитывает захардкоженную высоту под новую шапку.

**Tech Stack:** React 18, TypeScript (strict), TanStack Router, Tailwind + CSS-токены, lucide-react, react-i18next, UI Kit 2.0 (`dropdown-menu`, `avatar`).

## Global Constraints

- Работать строго внутри `frontend/`. Не трогать backend, БД, auth, routes, root-конфиги.
- Не выдумывать функционал сверх ТЗ. Поиск — presentational (без новой логики).
- Не добавлять роут `/settings`; пункт «Настройки» опустить, зафиксировать в `backend-endpoints-needed.md`.
- Использовать только существующие дизайн-токены и shared UI-компоненты, без локальных стилей сверх необходимого.
- Top bar — только desktop (`hidden lg:flex`). Mobile `MobileHeader` + `BottomNav` не трогать.
- Avatar-меню открывается по hover **и** click/focus.
- Sidebar не содержит Профиль / Уведомления / Подписка как nav-пункты; блок подписки — компактный статус без кнопки «Управлять».
- Меню аккаунта: Мой профиль → `/profile`, Мои объявления → `/my-ads`, Подписка → `/subscription`, Выйти.
- Sidebar-разделы в порядке: Лента, Каталог объявлений (`/ads`), Разместить объявление (`/ads/new`), Мои объявления (`/my-ads`, только авторизованным), Сообщества, Каналы, Мессенджер, Друзья.
- CSS-переменная `--desktop-topbar-h: 60px`.
- После изменений `npx tsc --noEmit` должен быть чистым.
- Не коммитить объединение/мерж без разрешения; коммиты по таскам разрешены.

**Проект не имеет unit-тест-раннера для компонентов (Vitest/RTL не настроены).** Поэтому «тест» каждой задачи — статическая проверка `npx tsc --noEmit` (из `frontend/`) плюс указанные grep-проверки. TDD в форме «сначала падающий тест» здесь заменяется на «сначала проверка, что цель ещё не выполнена (grep пустой / tsc-ошибка отсутствия), затем реализация, затем tsc-зелёный + grep подтверждает».

Для hover+click: держим `open` в state, открываем по `onMouseEnter` обёртки и по клику триггера (Radix сам обрабатывает клик через `onOpenChange`), закрываем по `onMouseLeave` с задержкой ~150ms. Avatar-паттерн (из `profile.tsx`): `<Avatar>` c `<AvatarImage src>` только если src непустой, иначе `<AvatarFallback>` с инициалами.

Создать `frontend/src/components/layout/UserMenu.tsx`:

```tsx
import { useRef, useState } from "react";
import { Link } from "@tanstack/react-router";
import { User, ClipboardList, Crown, LogOut } from "lucide-react";
import { useTranslation } from "react-i18next";
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuSeparator,
} from "@/components/ui/dropdown-menu";
import { Avatar, AvatarImage, AvatarFallback } from "@/components/ui/avatar";
import { useStore, selectors } from "@/lib/store";
import { signOut } from "@/lib/auth/session";
import { ROUTES } from "@/lib/routes";

function initials(name: string): string {
  const parts = name.trim().split(/\s+/).filter(Boolean);
  if (parts.length === 0) return "?";
  if (parts.length === 1) return parts[0].slice(0, 2).toUpperCase();
  return (parts[0][0] + parts[1][0]).toUpperCase();
}

export function UserMenu() {
  const me = useStore(selectors.currentUser);
  const { t } = useTranslation();
  const [open, setOpen] = useState(false);
  const closeTimer = useRef<ReturnType<typeof setTimeout> | null>(null);

  const cancelClose = () => {
    if (closeTimer.current) {
      clearTimeout(closeTimer.current);
      closeTimer.current = null;
    }
  };
  const scheduleClose = () => {
    cancelClose();
    closeTimer.current = setTimeout(() => setOpen(false), 150);
  };

  const hasAvatar = Boolean(me.avatar && me.avatar.trim());

  const handleSignOut = () => {
    void signOut().then(() => {
      window.location.href = "/login";
    });
  };

  return (
    <div onMouseEnter={() => { cancelClose(); setOpen(true); }} onMouseLeave={scheduleClose}>
      <DropdownMenu open={open} onOpenChange={setOpen}>
        <DropdownMenuTrigger asChild>
          <button
            type="button"
            aria-label={t("nav.profile")}
            className="grid place-items-center rounded-full outline-none transition-colors"
            style={{ width: 40, height: 40 }}
          >
            <Avatar className="h-9 w-9">
              {hasAvatar && <AvatarImage src={me.avatar} alt="" className="object-cover" />}
              <AvatarFallback
                className="text-[13px] font-bold"
                style={{ background: "var(--accent-soft)", color: "var(--accent)" }}
              >
                {initials(me.name)}
              </AvatarFallback>
            </Avatar>
          </button>
        </DropdownMenuTrigger>
        <DropdownMenuContent
          align="end"
          sideOffset={8}
          className="w-56"
          onMouseEnter={cancelClose}
          onMouseLeave={scheduleClose}
        >
          <DropdownMenuItem asChild>
            <Link to={ROUTES.profile} className="flex items-center gap-2">
              <User className="h-4 w-4" /> {t("nav.profile")}
            </Link>
          </DropdownMenuItem>
          <DropdownMenuItem asChild>
            <Link to={ROUTES.myAds} className="flex items-center gap-2">
              <ClipboardList className="h-4 w-4" /> {t("nav.myAds")}
            </Link>
          </DropdownMenuItem>
          <DropdownMenuItem asChild>
            <Link to={ROUTES.subscription} className="flex items-center gap-2">
              <Crown className="h-4 w-4" /> {t("nav.subscription")}
            </Link>
          </DropdownMenuItem>
          <DropdownMenuSeparator />
          <DropdownMenuItem onSelect={handleSignOut} className="flex items-center gap-2">
            <LogOut className="h-4 w-4" /> {t("auth.logout")}
          </DropdownMenuItem>
        </DropdownMenuContent>
      </DropdownMenu>
    </div>
  );
}
```

`lg:h-[calc(100vh-120px)]` — магическая 120px, рассчитанная на прежний desktop-шелл без топбара. ```

Новый desktop-хром сверху = топбар 60px + `<main>` `pt-4` (16px) + запас на нижний отступ/границы (~44px) ≈ прежним 120px, но теперь топбар выражен переменной.
