# Чек-лист ручного тестирования — neeklo.modelizmclub.ru

Демо-версия на mock-данных. Тестировать на **мобильном (реальный телефон)** и десктопе.
Логин не нужен — demo-режим включён автоматически.

## 0. Общее
- [ ] Сайт открывается быстро на мобильном интернете (3G/слабый Wi-Fi).
- [ ] Нигде нет горизонтальной прокрутки (страница не «ездит» вбок).
- [ ] Нижняя навигация (Лента/Сообщества/Сообщения/Объявления/Профиль) видна и работает на всех экранах.

## 1. Лендинг (главная, `/`)
- [ ] Кнопка «Открыть демо» в шапке компактная, помещается, бургер (☰) не за краем.
- [ ] Бургер-меню открывается, пункты + язык + «Войти» внутри экрана.
- [ ] На мобильном hero показывает картинку (не грузит тяжёлое видео); на десктопе — видео.
- [ ] Кнопки «Смотреть объявления» / «Открыть демо» не слишком большие.
- [ ] Секция «Экосистема для моделистов» → «Что есть в МоДелизМ».
- [ ] Категории показывают полные названия («Автомодели», «Железные дороги»), без обрезки.
- [ ] Переключение языка ru/en/zh меняет весь лендинг; после F5 язык сохраняется.

## 2. Каталог объявлений (`/ads`)
- [ ] Поиск + иконка фильтров в одну строку (по-авитовски), ниже — «Найдено: N» + сортировка.
- [ ] Чипсы категорий листаются вбок.
- [ ] Карточки в 2 колонки, сердечко и цена видны.
- [ ] Фильтры открываются по иконке, применяются, сбрасываются.
- [ ] Открытие объявления → галерея, «Написать продавцу», «В избранное», «Поделиться».

## 3. Избранное (`/favorites`)
- [ ] Иконка-сердечко в верхней панели на мобильном ведёт в Избранное.
- [ ] Добавленное сердечком объявление появляется здесь и переживает F5 (localStorage).

## 4. Мессенджер (`/messenger`)
- [ ] Список чатов: кнопка «+» и три точки на строках видны полностью (не обрезаны).
- [ ] Открытие чата: пузыри сообщений не вылезают за край, переносятся корректно.
- [ ] Шапка чата чистая: имя, статус, звонок, «…».
- [ ] Поле ввода, скрепка, фото, микрофон — на месте, клавиатура не ломает верстку.
- [ ] Плашка объявления в шапке (если писали продавцу с карточки) кликается.

## 5. Лента (`/feed`)
- [ ] Посты, лайки, комментарии, репост — открываются.
- [ ] Картинки постов подгружаются по мере прокрутки (lazy).
- [ ] Правый блок «Категории» с раскрытием подкатегорий (desktop).

## 6. Профиль (`/profile`)
- [ ] Обложка, аватар, статистика, вкладки (Публикации/Объявления/…) листаются.
- [ ] «Редактировать», смена фото (загрузить/удалить), «Выйти».

## 7. Сообщества / Каналы
- [ ] Списки открываются, обложки подгружаются, вступление/переход работают.

## 8. Адаптив / устройства
- [ ] iPhone SE (375px) и крупный телефон (414px) — без обрезаний.
- [ ] Планшет (768px) — раскладка перестраивается.
- [ ] Десктоп (1440px) — сайдбар, правые панели, топ-бар.
- [ ] Тёмная/светлая тема (переключатель в меню «…» / футере лендинга).

## Известные ограничения (демо, НЕ баги)
- Данные сбрасываются при полной перезагрузке (кроме избранного — оно в localStorage).
- Реальной загрузки файлов/аватара на сервер нет (демо-превью); нужен backend (см. `backend-endpoints-needed.md`).
- Плашка объявления в чате не сохраняется после перезагрузки (нужен `conversation.listing_id` на бэке).
- Видеозвонки/пуши работают только с реальным backend (не в demo).
